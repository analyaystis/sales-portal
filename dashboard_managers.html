Как считаются показатели

Этика / Гринвуд: KPI = 0,20×C1/С̄1 + 0,25×C2/С̄2 + 0,25×C3/С̄3 + 0,10×C4/С̄4 + 0,20×C5/С̄5
ДОК: KPI = 0,30×C1/С̄1 + 0,25×C2/С̄2 + 0,25×C3/С̄3 + 0,10×C4/С̄4 + 0,10×C5/С̄5
C1 = Встречи / Клиенты C2 = Брони / Клиенты C3 = Покупатели / Брони C4 = Нецелевые / Клиенты C5 = Целевые / Клиенты
Кнагруз = Клиенты / Ср.клиентов Кактив = 0,5×(Зв/Ср.зв) + 0,5×(Встр/Ср.встр) Крез = 0,3×(Нецел/Ср.нецел) + 0,7×(Цел/Ср.цел) Кприсут = Отраб.дни / Раб.дни
КТУ = KPI × (0,4×Кнагруз + 0,3×Кактив + 0,3×Крез) / (0,5 + 0,5×Кприсут)
Накопительные показатели

Звонки, Клиенты, Встречи, Брони, Покупатели — сумма. Конверсии, KPI, коэффициенты, КТУ — среднее.
